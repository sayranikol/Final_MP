SkillQuest Binary Tree Explorer

Open index.html in any modern browser.

Main files:
- index.html: UI layout
- styles.css: visual design and responsive styling
- script.js: binary tree data structure and app logic
- BinaryTree_SkillQuest_Report.pdf: project description with screenshots

How it uses a binary tree:
- Each quest has a numeric score/key.
- Lower scores are inserted into the left child.
- Higher scores are inserted into the right child.
- Search follows the binary-search-tree path.
- Inorder, preorder, and postorder traversals are shown as output.
